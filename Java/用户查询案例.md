# 用户查询案例

*   Servlet + JSP + MySQL + JDBCTemplate + Druid + BeanUtils + Tomcat
*   Web层、Service层、Dao层的架构协作

## 建立数据库

```sql
use test; 			 -- 使用数据库
create table user(   -- 创建表
	id int primary key auto_increment,
	name varchar(20) not null,
	gender varchar(5),
	age int,
	address varchar(32),
	qq	varchar(20),
    email varchar(50)
	);
```
